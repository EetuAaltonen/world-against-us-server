## Singleplayer

## NodeJS Server Setup

<br>

Create .env config file with parameters:
```
ADDRESS="127.0.0.1"
PORT=54544
```
<br>

Install node_modules
```
npm install
```

Navigate to ./src and run server with
```
npm run start
```
or
```
npm run dev
```
